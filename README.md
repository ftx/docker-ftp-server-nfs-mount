## FTP Server with NFS Mount 


##### FTP
|ENV Variable  |Required |Default   |Example   |
|---|---|---|---|
|FTP   |NO   |**NO**   |YES   |
|FTP_LOGIN   |NO   |flotix   |ftx   |
|FTP_PASSWD   |NO   |**CHANGEME**   |lsdfgzeihti$!976756   |

##### NFS
|ENV Variable  |Required |Default   |Example   |
|---|---|---|---|
|NFS_SERVER   |NO   |**NO**   |YES   |
|NFS_REMOTE   |YES   |/nfs   |/var/www   |
|NFS_LOCAL   |YES   |/nfs   |/var/www   |
